# GITHUB USER SEARCH

Простое веб-приложение для поиска пользователей на GitHub и их репозиториев.

## Функционал

- Поиск пользователя
- Просмотр профиля
- Просмотр публичных репозиториев
- Переход в репозиторий на GitHub

## Технологии

- React
- TypeScript
- Vite
- Material UI

## Запуск проекта

- npm install
- npm run dev

## Деплой

Проект развёрнут на Vercel.
